# Number Guessing Game

A small Java CLI project where the player guesses a random number and receives higher/lower hints.

## Run

```bash
javac src/Main.java
java -cp src Main
```

## Features

- Generates a number from 1 to 100.
- Gives higher or lower hints.
- Limits the player to 7 attempts.
- Prints a win or loss message with the target number.
