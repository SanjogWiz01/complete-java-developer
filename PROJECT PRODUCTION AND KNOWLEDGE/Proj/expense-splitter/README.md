# Expense Splitter

A small Java CLI project that splits a bill between people and optionally includes a tip.

## Run

```bash
javac src/Main.java
java -cp src Main
```

## Features

- Reads bill subtotal.
- Reads tip percentage.
- Reads number of people.
- Calculates tip amount, final total, and per-person share.
- Validates basic positive inputs.
