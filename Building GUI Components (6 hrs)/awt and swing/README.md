# AWT & Swing Comprehensive Tutorial

This project covers everything about Java AWT (Abstract Window Toolkit) and Swing GUI frameworks.

## Project Structure

### AWT Foundation (01-10)
- Basic Components and Containers
- Layout Managers
- Event Handling
- Graphics & Painting

### Swing Components (11-25)
- JFrame, JPanel, JButton
- Text Components
- List, Table, Tree
- Dialogs and Windows

### Advanced Swing (26-35)
- Look and Feel
- Threading & SwingWorker
- Document Models
- Custom Rendering

### Projects & Examples (36-50)
- Calculator
- Text Editor
- Todo Application
- Media Player
- Complete GUI Applications

## Prerequisites
- Java 8 or higher
- Basic Java knowledge

## Compilation & Running
```bash
javac *.java
java ClassName
```
