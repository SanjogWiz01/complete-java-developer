About CabBook

CabBook is a modern online cab booking system built with Spring Boot and advanced technologies.

Features

  • Real-time ride tracking
  • Transparent pricing with no hidden charges
  • Verified and experienced drivers
  • Multiple vehicle options
  • Rating and review system
  • Secure payment processing

Technology Stack

  • Backend: Spring Boot, Spring Security, Hibernate ORM
  • Frontend: Thymeleaf, Bootstrap, HTML5/CSS3
  • Database: MySQL
  • Location Services: Haversine Formula for distance calculation
