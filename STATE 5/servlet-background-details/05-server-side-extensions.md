# Need for Server Side Extensions

Server-side extensions allow a web server to generate dynamic output instead of returning only static files.

Servlets are one Java-based server-side extension technology.

## Static vs Dynamic

Static page:

- Same file returned every time.
- Usually HTML, CSS, JavaScript, or images.

Dynamic servlet response:

- Output can change based on user input.
- Can read databases, sessions, and request parameters.
- Can apply validation and business rules.

## Example Use Cases

- Login form processing.
- Product catalog pages.
- Booking systems.
- Admin dashboards.
- API endpoints returning JSON.
