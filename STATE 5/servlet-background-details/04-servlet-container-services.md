# Servlet Container Services

A servlet container provides runtime support for Java web applications.

Apache Tomcat is a common servlet container used for learning and production-style practice.

## Services Provided

- Loads and initializes servlet classes.
- Maps URLs to servlet classes.
- Creates request and response objects.
- Manages servlet lifecycle methods.
- Handles concurrent requests with threads.
- Encodes and decodes HTTP/MIME data.
- Manages sessions with cookies or URL rewriting.
- Applies security rules and authentication constraints.

## Why It Matters

Without a container, every servlet application would need custom networking, routing, threading, and lifecycle code.
