# Introduction to Java Servlets

A Java Servlet is a server-side Java class used to build dynamic web applications.

Unlike a static HTML page, a servlet can read request data, call Java services, access a database, and generate a response at runtime.

## Key Points

- Runs inside a servlet container.
- Handles HTTP requests and responses.
- Supports enterprise web application development.
- Commonly used with JSP, JDBC, filters, and MVC patterns.

## Where Servlets Fit

Client browser -> Web server -> Servlet container -> Servlet -> Response back to browser

Servlets are not executed directly by the user. They are managed by the container.
