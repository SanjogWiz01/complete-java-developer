# Servlet Background Details

This folder expands the `STATE 5/background` notes into focused learning files for Java Servlets.

Servlets are server-side Java components that receive client requests, process business logic, and return dynamic responses through a servlet container such as Apache Tomcat.

## Study Path

1. Start with the servlet introduction.
2. Understand the request-response lifecycle.
3. Review the servlet architecture and container responsibilities.
4. Practice a basic servlet program.
5. Learn deployment, session, MIME, and security details.

## Folder Goal

The notes are written as quick revision material for building Java web applications using Servlets, JSP, and Tomcat.
