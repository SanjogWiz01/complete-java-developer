# Nepal Bachelor IT Education Gap

This class notes webpage discusses the lack of practical, industry-ready bachelor level IT education in Nepal.

## Page Purpose

- Explain the gap between academic theory and real software work.
- Highlight common issues such as outdated lab work, low project feedback, weak industry connection, and limited documentation practice.
- Provide a semester-wise improvement roadmap.
- Give students a small checklist for building stronger practical habits.

## Run

Open `index.html` in a browser, or serve this folder with a local static server.

## Discussion Questions

- What practical skill is missing most in your current IT classes?
- Which project would best prove that a bachelor student is industry ready?
- How can colleges connect students with mentors, alumni, and local companies earlier?
