## 📖 About This Repository

This repository chronicles my learning journey through the **Advanced Programming with Java** course at Pokhara University, Faculty of Science and Technology. It serves as both a personal knowledge base and a resource for fellow learners exploring Java development from fundamentals to advanced enterprise applications.

## 🎯 Learning Objectives

- ✅ Master Java programming fundamentals and architecture
- ✅ Deep dive into Object-Oriented Programming principles
- ✅ Build interactive GUI applications using Swing and JavaFX
- ✅ Develop networked and distributed applications
- ✅ Implement database connectivity with JDBC
- ✅ Create dynamic web applications using Servlets and JSP
- ✅ Explore enterprise frameworks (Hibernate, Spring Boot)
- ✅ Understand concurrency, multithreading, and design patterns

## 📚 Course Structure

### **Unit 1: Basics of Programming in Java** (7 hrs)
- Java Architecture & Sample Programs
- Classes, Objects, and Constructors
- Packages and Data Types
- Conditional Statements
- Access Modifiers
- Exception Handling
- Java Collections Framework

### **Unit 2: Object-Oriented Principles** (6 hrs)
- OOP Fundamentals Review
- Inheritance (Types, Super & Extends keywords)
- Method Overriding & Overloading
- Abstract Classes and Interfaces
- Upcasting vs Downcasting
- Final Classes and Methods

### **Unit 3: Building GUI Components** (6 hrs)
- AWT and Swing Fundamentals
- Layout Management
- GUI Controls and Menu Elements
- Event Handling & Listeners
- JavaFX vs Swing
- JavaFX Layouts and UI Controls

### **Unit 4: Distributed Network Programming** (8 hrs)
- TCP, UDP, IP Addressing
- Socket Programming
- URL Handling
- Email Integration (JavaMail API)
- RMI (Remote Method Invocation)
- CORBA Architecture
- IDL and Distributed Systems

### **Unit 5: Database Connectivity** (5 hrs)
- JDBC Architecture & Driver Types
- Connection Management
- SQL Operations (DDL & DML)
- Prepared Statements & SQL Injection Prevention
- Result Sets and Transactions
- RowSets and SQL Escapes

### **Unit 6: Servlets and JSP** (6 hrs)
- Web Application Overview
- HTTP Methods and Responses
- Servlet Lifecycle
- Form Processing (GET/POST)
- Database Connectivity through Servlets
- Session Management & Cookies

### **Unit 7: Advanced Topics** (7 hrs)
- Object-Relational Mapping (ORM)
- Hibernate Framework
- Spring Boot Basics
- Concurrency & Multithreading
- Design Patterns (Singleton, Factory, Abstract Factory)

## 💡 How to Use This Repository

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/advanced-java-journey.git
   cd advanced-java-journey
   ```

2. **Navigate to specific topics**
   ```bash
   cd 01-basics
   ```

3. **Run examples**
   ```bash
   javac YourProgram.java
   java YourProgram
   ```

4. **For Maven projects**
   ```bash
   mvn clean install
   mvn exec:java
   ```
