# Daily BMI Checker

A small command-line Java project that calculates body mass index from weight and height, then prints the matching health category.

## Run

```bash
javac src/Main.java
java -cp src Main
```

## Features

- Accepts weight in kilograms.
- Accepts height in centimeters.
- Shows BMI rounded to one decimal place.
- Classifies the result as underweight, normal, overweight, or obese.
