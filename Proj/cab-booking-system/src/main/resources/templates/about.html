CabBook for Pokhara

A Spring Boot cab booking system with rider accounts, map-based trip creation, fare estimates, driver assignment, and admin operations.

Book a Ride
Map booking

Leaflet maps use Pokhara defaults and route previews.

Fare controls

Fare logic uses a clear Rs. 50/km distance rate with promo code support.

Admin workflow

Admins can assign drivers, start rides, complete rides, and inspect users.

Technology Stack

Backend
Spring Boot 3
Security
Spring Security
Database
H2 or MySQL
Frontend
Thymeleaf, Bootstrap, Leaflet
