# Temperature Unit Converter

A command-line Java project that converts a temperature between Celsius, Fahrenheit, and Kelvin.

## Run

```bash
javac src/Main.java
java -cp src Main
```

## Features

- Reads a numeric temperature.
- Accepts source and target units as `C`, `F`, or `K`.
- Validates unit names.
- Prints the converted value rounded to two decimal places.
