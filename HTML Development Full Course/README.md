# HTML Development Full Course

This folder contains a complete HTML learning path from beginner fundamentals to
project-ready page structure. The course focuses on writing clean, semantic,
accessible HTML that can later be styled with CSS and connected to JavaScript.

## Files

- `HTML_DEVELOPMENT_FULL_COURSE.md`: Main course notes, examples, practice tasks,
  mini projects, and final capstone requirements.

## Learning Goals

After completing this course, you should be able to:

- Explain how HTML fits into the web development workflow.
- Build a valid HTML document from scratch.
- Use headings, paragraphs, lists, links, images, tables, and forms correctly.
- Structure pages with semantic elements such as `header`, `nav`, `main`,
  `section`, `article`, `aside`, and `footer`.
- Write accessible markup with useful labels, alt text, document language, and
  logical heading order.
- Prepare HTML pages for CSS styling, JavaScript behavior, search engines, and
  deployment.
- Build complete pages such as a profile page, blog article, registration form,
  product page, and portfolio website.

## Suggested Study Plan

| Day | Topics |
| --- | --- |
| 1 | Web basics, document structure, text content |
| 2 | Links, images, media, semantic layout |
| 3 | Tables, forms, validation |
| 4 | Accessibility, SEO, CSS/JavaScript integration |
| 5 | Mini projects and capstone portfolio |

## How to Practice

1. Create a new `.html` file for each lesson.
2. Type the examples manually instead of only reading them.
3. Open the file in a browser and inspect it with developer tools.
4. Validate the page structure and fix one issue at a time.
5. Complete every exercise before moving to the mini projects.

## Final Output

The final capstone is a multi-section personal portfolio page that includes:

- A semantic page layout.
- Navigation links.
- About, skills, projects, education, and contact sections.
- Accessible images and form fields.
- Clean metadata for browser tabs and search engines.
